Lesson 01

JavaScript Basics before ee.* objects

Learning goals

  • Verify the Earth Engine Code Editor works
  • Understand how print() outputs information
  • Learn variables, arrays, and dictionaries
  • Create a simple function

Concept

Google Earth Engine uses JavaScript as the interface language. Before analyzing satellite imagery, we first practice basic programming structures. All Earth Engine objects (ee.Image, ee.FeatureCollection, ee.ImageCollection) behave similarly to JavaScript objects.

This lesson prepares you to read and understand every script used in later remote sensing exercises.
