Google Earth Engine Workshop

Wall-to-wall forest biomass estimation

Workshop description

Forest biomass is a key indicator for estimating forest carbon stocks. Biomass maps help identify carbon pools, explore carbon credit opportunities, and support biodiversity conservation and ecosystem protection.

This hands-on workshop teaches a complete wall-to-wall biomass mapping workflow in Google Earth Engine (GEE). Participants will prepare biomass reference data using GEDI (Gridded Aboveground Biomass Density), train models, generate a continuous biomass map, evaluate accuracy, interpret spatial patterns, and export final products.

Learning objectives

  • Write and understand code in Google Earth Engine
  • Access GEDI and satellite imagery datasets
  • Design samples and build prediction models
  • Produce wall-to-wall biomass maps
  • Evaluate and interpret spatial results

Target audience

Students, researchers, and professionals in forestry, ecology, remote sensing, and environmental sciences interested in forest monitoring and geospatial analysis. Basic familiarity with Earth Engine coding is recommended.

Expected outcomes

  • Produce a forest biomass map using GEE
  • Use cloud-based geospatial analysis workflows
  • Apply methods to ecological monitoring and resource management
