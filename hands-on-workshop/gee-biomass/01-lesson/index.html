GEE Lesson 01

JavaScript Basics (before ee.*)

Learning goals

  • Verify the GEE Code Editor works
  • Use print() to show output
  • Understand variables, arrays, objects
  • Write a simple function

Explanation

Google Earth Engine uses JavaScript in the Code Editor. Before working with satellite data (ee.Image, ee.ImageCollection, etc.), we practice basic JavaScript structures that are required for every GEE workflow.
